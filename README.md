## Installation

Run:

```
gem install tomo-plugin-goodjob
```

Or add it to your Gemfile:

```ruby
gem "tomo-plugin-goodjob"
```

Then add the following to `.tomo/config.rb`:

```ruby
plugin "goodjob"
```
